## VIM SETTINGS

before setting, you must install Vundle.

  `$git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim`

Then, install plug in by opening vim

  `vi`
  
Now, just type

  `PluginInstall`
 


for neocomplete, see

[neocomplete](https://github.com/Shougo/neocomplete.vim)
